About the Project

Project Title

A Machine Learning-Based Real-Time Remaining Useful Life Estimation and Fair Pricing Strategy for Electric Vehicle Battery Swapping Stations

Project Overview

This project aims to estimate the Remaining Useful Life (RUL) of EV batteries using machine learning and suggest fair pricing based on battery health. The system leverages real-world operational data to drive smarter decisions in the EV battery swapping domain.

Machine Learning Models

Starting with XGBoost, we enhanced performance with Random Forest Regressor, KNN Regressor, Stacking, and Voting Regressors. These models provide accurate RUL predictions and help in dynamically adjusting swap pricing.

Why It Matters

By offering predictive insights and fair pricing strategies, the system empowers users and station operators to make informed choices, improve cost efficiency, reduce battery waste, and maintain optimal performance across EV battery inventories.

Interactive Web Platform

The system features a user-friendly web interface for real-time prediction and pricing suggestions. Users can input battery parameters and instantly receive insights, enhancing trust and transparency in EV infrastructure management.
